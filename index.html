En underrubrik

En kort och väldigt opersonlig presentationstext om mig själv.

Jag gjorde min första hemsida i slutet av 90-talet. Då gjordes layout med tabeller och färgade bakgrundsbilder. Det häftigaste man kunde göra var en animerad GIF. Sen kom CSS och med DIV och float gick det att göra sidor med betydligt mer avancerad layout.

The holy grail var nu inom räckhåll. Men mycket tid fick läggas på att hitta sätt att hantera webbläsarnas olika implementering av CSS och en stor del av webbdesign handlade om att göra avancerad grafik i program som Photoshop.

Smart phones gjorde att Responsive Web Design blev ett måste. CSS utvecklades och kompletterande teknik som preprocessors och frameworks började dyka upp.

Introduktionen av CSS grid har äntligen gjort det möjligt att få till en flytande rutnätsbaserad layout på skärmen. Som med alla nya funktioner har det tagit några år innan det slagit igenom men stöds nu av de flesta webbläsare.

Detta är första gången jag gör en hemsida med CSS grid, flexbox och de nya HTML-elementen som introducerades med HTML5, header, nav, main, section och footer.
